# Customer 360 Marketing Campaign Analytics

## Project Overview

This project demonstrates a complete Data Analytics workflow using PostgreSQL, SQL, and Power BI. The objective was to analyze customer demographics, purchasing behavior, campaign performance, and product spending to generate actionable business insights.

---

## Tools & Technologies

- PostgreSQL
- SQL
- Power BI
- Microsoft Excel

---

## Dataset Information

- Dataset Name: Customer_360_Marketing_Campaign_Analytics
- Total Records: 1001
- Total Columns: 37

---

## Project Workflow

1. Data Import into PostgreSQL
2. Data Cleaning
3. Data Transformation
4. SQL Analysis
5. Dashboard Development in Power BI
6. Business Insights & Recommendations

---

## SQL Analysis

The project includes analysis of:

- Total Customers
- Total Spending
- Average Income
- Average Spending
- Product Spending
- Purchase Channel Analysis
- Country Analysis
- City Analysis
- Customer Segmentation
- Campaign Performance
